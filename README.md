# ST.kg — СпецТех KG 🏗️

Платформа для аренды спецтехники в Кыргызстане с ИИ-помощником.

## Возможности:
- Заказ техники во всех 7 областях КР.
- ИИ-чат для подбора комплектов техники.
- Регистрация партнеров (водителей) с фото и характеристиками.
- Интеграция с Telegram и CRM.
